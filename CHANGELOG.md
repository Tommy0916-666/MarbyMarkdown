# Changelog

All notable changes to this project will be documented in this file.

## \[0.1.1] - 2026-04-26

基于 [Milkup](https://github.com/Auto-Plugin/milkup) 开源项目的第一版二次开发发布！

### Added

#### 文档与教程

- 新增 `GUIDE.md` 完整使用指南，包含：
  - 软件详细使用教程
  - Markdown 基本语法教程

#### 设置页面重构

- 新增 **"拓展功能"** 设置页面
- 新增 **"最近文件"** 功能：
  - 在主界面侧边栏显示最近打开的文件
  - 可自定义最大记录数量
- 新增 **"自动保存"** 功能：
  - 可选择是否启用自动保存
  - 自定义保存间隔（秒）
  - 支持文件自动回退（防止误操作）
  - 未命名文档自动保存时的智能提示弹窗
- 新增 **"文件恢复"** 功能：
  - 自动打开上次关闭时正在编辑的文件

#### AI 功能全面升级

- **AI 配置架构重构**：
  - 全局 AI 配置基础
  - 每个 AI 功能支持独立配置（可选是否启用独立配置）
- **AI 续写**增强：
  - 新增手动触发模式
  - 新增接受续写快捷键
  - 支持自定义快捷键
- **AI 优化**新增功能：
  - 右键菜单直接触发优化
  - 选择优化风格
  - 自定义优化提示词
  - AI 修改后自动替换选中的文本
- **AI 聊天**全新开发：
  - 侧边栏式聊天界面
  - 支持多会话管理
  - 每个会话强关联当前文件
  - 历史记录查看与管理
  - 新建会话功能
  - 只读模式（查看其他文件的历史会话）
  - 全屏模式（流畅动画）
  - 支持 Markdown 渲染（用户和 AI 消息）
  - GUIDE 大纲适配
  - AI 自动生成会话标题
  - "回到最下方"悬浮按钮

#### 界面优化

- 左下角：新增文档保存状态提示（图标+文字）
- 页尾中间：显示 MarbyMarkdown 名称+版本号
- 页尾右侧：自动保存开关（可点击切换，显示当前间隔）
- 页尾：新增 AI 聊天按钮（需先在设置启用）
- 关于页面：更新图标、版权信息（保留原作者+新增自己）
- 外观设置：编辑器描述改为 MarbyMarkdown

### Changed

- 将 AI 续写从设置中移到"拓展功能"页面
- 优化侧边栏打开/关闭动画
- 改进全屏模式体验
- 文档文件（README/GUIDE/CHANGELOG）打包到安装目录根目录
- 开发者工具：只在开发模式下自动打开，打包后禁用 F12 快捷键

### Fixed

- 修复切换文档时会话混乱的问题
- 修复 macOS 构建失败的问题
- 修复开发者工具在打包后自动打开的问题
- 修复各种 UI 显示问题
- 完善多平台打包（Windows/macOS/Linux）
